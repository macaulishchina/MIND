# Verification Profile: Feature

Use this profile for changes that introduce or alter user-visible behavior.

## Required Checks

- `spec-consistency`
- `workflow-integrity`
- `change-completeness`
- `manual-review`

## Typical Use

- new capability
- behavior-changing enhancement
- bugfix that changes accepted behavior or acceptance criteria

## Success Standard

The proposal, spec delta, implementation tasks, and verification evidence all
align with the intended user-visible outcome.
