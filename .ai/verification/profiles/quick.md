# Verification Profile: Quick

Use this profile for narrow, low-risk changes that still need a change folder.

## Required Checks

- `workflow-integrity`
- `change-completeness`
- `manual-review`

## Typical Use

- small bugfix with localized impact
- wording or docs change that still merits a proposal
- narrow behavior update with obvious validation

## Success Standard

The change is internally consistent, clearly documented, and manually verified
at the level needed for the requested scope.
