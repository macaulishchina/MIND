# Check: manual-review

## Purpose

Provide a universal fallback check when automation is absent or incomplete.

## When To Use

- every current change in this repository until stronger automation exists

## Pass Criteria

- The implementer reviewed the changed artifacts end to end.
- The chosen verification profile is addressed with explicit notes.
- Any unverified area is clearly called out as residual risk.

## Acceptable Evidence

- written walkthrough
- scenario notes
- reviewer signoff
